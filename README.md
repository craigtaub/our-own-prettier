# our-own-prettier

A simple VSCode extension which auto-formats 1 scenario.

## Scenario 1

### Before

```javascript
myFunction();
```

### After

```javascript
myFunction();
```

## Scenario 2

### Before

```javascript
myFunction(one, two, three);
```

### After

```
myFunction(
  one,
  two,
  three
);
```
